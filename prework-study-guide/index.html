HTML

  • The head element contains information about the webage. This information includes metadata
  • The body element represents the visible contenet shown to the user
  • Elements in HTML are case sensitive
  • An element refers to the entire element from opening tag to closing tag, whereas a tag refers only to what appears inside the angle brackets.
  • Opening and closing tags are represented by angle brackets

CSS

  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • To style an element directly in the HTML file use inline CSS. Inline styles apply to a specific HTML tag, using a style attribute with a CSS rule to style a particular page element only
  • Do Not Repeat Yourself (DRY)

GIT

  • A version control system that lets you manage and keep track of each version of your project
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone [url]: retrieve an entire repository from a hosted location via URL
  • git log: show all commits in the current branch’s history
